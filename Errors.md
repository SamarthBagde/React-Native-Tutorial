## 1. SDK location not found.

- This error occure because `local.properties` is missing inside andriod folder
- Solution : If that fill is missing then create file with name `local.properties` and add below line

- For andriod 
``` sdk.dir=C:\\Users\\YourUsername\\AppData\\Local\\Android\\Sdk ```

- For ios 
``` sdk.dir = /Users/apple/Library/Android/sdk ```
